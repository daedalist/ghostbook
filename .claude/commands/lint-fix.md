Run ESLint with automatic fixing enabled to clean up code style issues.

Steps:

1. Run `npm run lint:fix` to automatically fix ESLint issues
2. Report what issues were fixed
3. If there are any remaining issues that couldn't be auto-fixed, list them with their locations
4. Suggest fixes for any remaining issues
